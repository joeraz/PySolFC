Wizard's Castle

Hex A Deck type. 1 deck. No redeal.

Object

Move all cards to the foundations.

Rules

Rows build down in rank in alternating color. Wizards are wild, so any card can be played on one, and a wizard can be played on any card. Only one card can be moved at a time. Foundations build up in rank by suit. Any card can be played on an empty row. Cards can be played from the foundations.
